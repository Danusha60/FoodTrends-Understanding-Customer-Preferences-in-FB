Nutrition, Lifestyle & Food Analytics Dashboard
📌 Project Overview
This project presents an interactive data visualization dashboard built using Power BI to analyze food preferences, nutrition patterns, lifestyle habits, and obesity-related indicators.

The dashboard provides insights into:
Dietary preferences and restaurant trends,Nutrition and physical activity patterns,Breakfast recommendations and nutrient comparision,Lifestyle impact on health metrics like BMI, cholesterol and disease risk

Objectives:
Analyze food consumption patterns and dietary choices
Understand the relationship between lifestyle and health indicators
Identify trends in restaurant preferences and ordering behavior
Provide meaningful insights for better health and nutrition decisions
